# Writing in a Ralph Wiggum Loop

Use the **writing-ralph-wiggum-loop** skill when you want to write iteratively until several conditions are met.

**Skill location:** `skills/writing-ralph-wiggum-loop/SKILL.md`

## Quick reference

- **Define conditions first** — explicit, checkable criteria (word count, coverage, style, etc.)
- **Loop:** write → check → feedback → iterate until all conditions pass
- **Stop only when** every condition passes or a safety limit is hit
